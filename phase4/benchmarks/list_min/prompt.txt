Write a Lean 4 specification for a function that returns the minimum element
of a list of natural numbers.

Requirements:
- Declare `listMin : List Nat → Nat` as an `axiom`.
- Define helper predicates as `def` with concrete recursive bodies.
- State theorems (using `:= sorry`) that fully specify minimum semantics.

The specification should be self-contained (no imports beyond Lean's core).

Properties that a strong specification should capture:
- `listMin xs` is a lower bound for every element of `xs`.
- `listMin xs` is itself an element of `xs` when `xs` is non-empty (otherwise
  the constant function `listMin xs = 0` also satisfies the lower-bound).

Output ONLY the Lean code.  Do not include explanations or commentary.
